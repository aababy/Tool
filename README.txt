1.基本步骤
	1. 导入主角
	1. 增加Atk,
	2. 调整坐标, 锚点..
	3. 导入特效
	4. 调整坐标, 锚点..
	5. 预览
	6. 保存

1.导入资源文件路径[允许]包含中文名(Mac上测试通过)
2.无论主角还是特效都是通过[导入]进行导入操作, 默认第1个导入的为主角, 需要时可以清空.
4.增加atk, 只会从最后1个开始增加(当前帧必须位于最后一个atk后面), 删除也是.
6.[导入]按钮, 第一次导入主角, 以后导入特效.
7.[保存]前, 先设置保存路径, 点[导入]->[修改保存路径]
